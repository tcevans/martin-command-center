# Martin's Command Center

A terminal dashboard for the orchestrator. Real-time visibility into agents, projects, and GitHub status.

## Features (Planned)
- Live agent status viewer
- Project health board
- GitHub activity feed
- Blocked items queue
- Auto-refresh (30s intervals)

## Tech Stack
- Python
- Textual (TUI framework)
- GitHub CLI integration
- OpenClaw API integration
